## 如何多开

您可以多开框架，但是需要保证每个框架监听的端口不一样，即可。

多开框架教程：

1.复制框架文件夹到别的地方

2.打开config/config.ini

3.修改[system]节点下的监听端口

4.保存，启动框架即可
